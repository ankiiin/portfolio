Auction House

Semester Project 2

About the project

Auction House is an auction platform designed to make bidding easy to understand, with clear listing cards, an intuitive flow, and a clean interface powered by API data.

My role

UX planning, information structure, interaction decisions, and frontend development with a focus on clarity, responsiveness, and accessibility.

Key reflections

In the final polishing phase, I improved the overall first impression by making the feed accessible for guests and adjusting navigation states based on login status. A natural next step would be to align the mobile hamburger menu states with the desktop navigation for a fully consistent experience.
